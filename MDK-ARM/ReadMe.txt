General Informatin about the board.
1. PC communication/Programming Port 					= UART01 
2. Serial Port for BLE & External Serial module			= UART02 
3. I2C Port for BME280, MS5611,MPU9250,TLV4930			= I2C1
4. SPI Port for External Flash Memory & ADXL345/375		= SPI1
5. Chip Select sigal for ADXL345 Memory					= SPI1-NSS1 (PA15)
6. Chip Select sigal for Flash Memory					= SPI1-NSS2 (PA8)
7. General Purpose LED ports							= GPLED1(PA0) & GPLED2(PA1)
8. External Trigger Points								= TRG01 (PB0) & TRG02 (PB1)
